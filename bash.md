## git

```
git add .

```

## ls
- `-a` all (include hidden)
- `-l` long (details)
- `-t` time (last modified)
- `-S` extension
- `-v` version sort
- `-S` size sort
- `-R` recursively
- `-1` ?

## psql
- `\dt` display public tables
- `\d [table]` display table schema

## venv
Virtual Environments
```
which python
python3 venv [directory]
source [directory]/bin/activate
deactivate
```
